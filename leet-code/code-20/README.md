# 题目

## 地址

[有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

## 思路

### 思路1

1. 判断长度是否为空
2. 判断长度是否是整数
3. 判断是否每一个字符都是标准字符
4. 设置两个数组，分别是左符号和右符号， 设置一个map，是右符号和左符号的映射
5. 遍历原始数组， 如果是左符号， 就放入到新数组，如果是右符号，就把新数组最后一个拿出来，看是否匹配， 如果是，那么继续， 不是就直接fasle
6. 如果循环走完了， 就需要判断新数组的长度是否Wie空， 如果不是， 那么就是有问题的
